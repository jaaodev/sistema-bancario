Este repositório contém o projeto de um sistema bancário que eu desenvolvi para aprimorar minhas habilidades com Java, POO, Exceções e Boas Práticas. 

##  Conceitos utilizados:

- Estruturas de repetição e estruturas condicionais.
- Scanner
- Métodos
- Associação
- Construtores
- Sobrecarga
- Encapsulamento
- Herança
- Abstração
- Polimorfismo
- Exceções

##  Estrutura do Projeto
O projeto está organizado por pacotes para facilitar a localização:

* **src/**: Onde estão localizados os pacotes do projeto.
    * `app` contém o inicializador do programa.
    * `domain` contém as informações bancárias.
    * `exception` contém as exceções aplicadas.
    * `service` contém o manager que é oque troca as informações entre as contas.
